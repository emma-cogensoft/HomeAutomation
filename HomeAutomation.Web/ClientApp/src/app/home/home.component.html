Home Energy Dashboard

This application displays real-time energy information for a home solar system — including battery state, solar input, inverter settings, and local weather — to help make informed decisions about energy usage.

Technologies:

  • ASP.NET Core 10 and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

External APIs:

  • Solax — cloud and local inverter API
  • Open-Meteo — free open-source weather API
