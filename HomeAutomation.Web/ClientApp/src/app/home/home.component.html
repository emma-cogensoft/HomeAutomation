Hello, world!

This application is designed to display battery and inverter settings information for a home solar system, using the SOLAX API.

Technologies:

  • ASP.NET Core 7 and C# 11 for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

External APIs:

  • SOLAX API - both cloud and local
  • Met Office API
